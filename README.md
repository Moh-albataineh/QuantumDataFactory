# Digital Data Factory

A computational chemistry project for molecular modeling, quantum chemical calculations, and data analysis. This project leverages Python and Psi4 for quantum mechanical simulations and electronic structure calculations.

## Getting Started

1. Activate the conda environment:
   ```bash
   conda activate ddf
   ```

2. Run your calculations as needed.
